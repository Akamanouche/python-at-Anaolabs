# Shortcuts courants Eclipse

	- ALT + Up/Down   : move line Up/Down 
	- ALT + SHIFT + R : refactoring
	- F3              : go to definition (e.g: ... of function)
	- ALT + SHIFT + L : extract local variable
	- ALT + SHIFT + M : extract method 
	

# CTRL + F9 ==> run tests

# CTRL + F11 ==> re-run last test
