
=======================================================================
Interpréteur de commandes (dit aussi "line-oriented command processor")
=======================================================================

On utilise ici l'objet "cmd.Cmd", qu'il suffit d'étendre pour fabriquer un interpreteur de commande
(tout comme dans le "BeeShell" de Beeware)


Liens
-----
	> http://docs.python.org/library/cmd.html
	> http://www.doughellmann.com/PyMOTW/cmd/index.html
	
	
Remarques
---------

	1) Pour sortir du shell, appuyer sur [CTRL + D]
	
	2) Il y a aussi un objet "shlex" pour faire de l'analyse syntaxique, et écrire des "mini-langages"
	
		> http://docs.python.org/library/shlex.html
